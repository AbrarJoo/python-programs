# Python Programs

This repository contains beginner Python programs created while learning Python fundamentals.

The programs focus on basic programming concepts such as:

* Variables and data types
* Conditional statements
* Loops
* Functions
* Simple problem-solving exercises

## Purpose

The goal of this repository is to practice and improve Python programming skills through small exercises and examples.

## Contents

Some of the programs in this repository include:

* Basic input/output programs
* Loop and condition examples
* Simple mathematical programs
* Beginner algorithm implementations

## How to Run

1. Make sure Python is installed on your system.
2. Clone the repository:

```
git clone https://github.com/AbrarJoo/python-programs.git
```

3. Navigate to the folder:

```
cd python-programs
```

4. Run any program:

```
python filename.py
```
